WinPcapANEDll
=============

**What is WinPcapANEDll**  
C dll of WinPcap wrapper for ActionScript Native Extension (ANE): WinPcapExtension.  
See the repository of [WinPcapExtension.](https://github.com/ryujimiya/WinPcapExtension/)
